## 为什么

当前账号管理的导入导出仅支持全量，无法按分组导出，也无法在导入时指定目标分组。这导致多分组管理时需要额外的手工筛选与分配，效率低且易出错。现在引入分组维度的导入导出能力，可以直接覆盖常见的运营和迁移场景。

## 变更内容

- 导出支持按分组选取（可多选/全选），导出结果仅包含所选分组的账号。
- 导入支持选择目标分组，导入的账号自动归入该分组。
- 若导入文件包含分组字段，提供一致的优先级规则（例如：显式选择的目标分组优先）。

## 功能 (Capabilities)

### 新增功能
- `account-group-import-export`: 账号管理导入导出支持按分组导出与导入指定分组。

## 影响

- 后端：导出接口增加分组选项；导入接口增加目标分组参数与校验；数据查询按分组过滤。
- 前端：账号管理导入/导出 UI 增加分组选择与提示；导入流程增加目标分组选择。
- 文档：导入导出字段与规则说明。
